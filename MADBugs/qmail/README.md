# qmail-remote RCE via DNS MX Hostname Shell Injection (CVE-2026-41113)

Remote code execution in [sagredo-dev/qmail](https://github.com/sagredo-dev/qmail) through shell injection in `tls_quit()`. An attacker who controls DNS for any domain the target server sends mail to can execute arbitrary commands as the `qmailr` user.
